SITE READY TO USE

Fișiere incluse:
- index.html
- despre.html
- servicii.html
- style.css
- script.js

Cum pui pozele tale:
1. Creezi un folder numit /images
2. Încarci pozele, de ex:
   - images/dr-gligor.jpg
   - images/cabinet-1.jpg
   - images/cabinet-2.jpg
3. Înlocuiești în fișiere linkurile de tip https://images.unsplash... cu căile locale:
   - url('images/dr-gligor.jpg')
   - <img src="images/cabinet-1.jpg" ...>

Unde sunt locurile pentru poze:
- Home: hero-card, photo-stage, gallery-grid
- Despre: photo-stage, gallery-grid

Site-ul este responsive și gata de publicare pe hosting static.
